# Super Store Sales Dashboard using Excel

## Project Overview
Created an interactive Excel dashboard to analyze Super Store sales data, profit trends, customer performance, and regional insights using charts and visualization techniques.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting
- Slicers

## Features
- Total sales and profit analysis
- Region-wise sales performance
- Category and sub-category insights
- Monthly sales trends
- Interactive dashboard filters

## Skills Gained
- Dashboard Creation
- Data Visualization
- Data Cleaning
- Business Analysis
- Analytical Thinking

## Project Outcome
The dashboard helps identify sales trends, profit performance, and business insights through interactive visual reports for better decision-making.
