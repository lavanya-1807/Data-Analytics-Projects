# Zomato Sales Analysis Dashboard - Power BI

## Project Overview
Created an interactive Power BI dashboard to analyze Zomato sales data, customer trends, restaurant performance, and order insights using data visualization techniques.

## Tools Used
- Power BI
- Data Visualization
- Dashboard Design
- Data Analysis

## Features
- Total sales and order analysis
- City-wise and state-wise performance
- Monthly sales trends
- Customer rating analysis
- Interactive filters and slicers
- KPI cards for revenue and orders

## Skills Gained
- Power BI Dashboard Creation
- Data Cleaning
- Business Intelligence
- Data Visualization
- Analytical Thinking

## Project Outcome
The dashboard helps understand sales performance, customer behavior, and restaurant insights through interactive visual reports for better business decision-making.
