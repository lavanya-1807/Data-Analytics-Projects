# Car Sales Data Analysis using SQL

## Project Overview
Performed data analysis on car sales datasets using SQL queries to identify sales trends, customer insights, and business performance.

## Tools Used
- SQL
- MySQL

## Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions
- JOINS

## Features
- Total car sales analysis
- Brand-wise sales performance
- Revenue analysis
- Customer purchase trends
- Monthly sales insights

## Skills Gained
- SQL Query Writing
- Data Analysis
- Data Filtering
- Database Management
- Analytical Thinking

## Project Outcome
The project helped generate meaningful business insights from structured car sales data using SQL queries and reporting techniques.
